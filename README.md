<h2>🛠️ Installation Steps:</h2>

<p>1. Install The repo</p>

<p>2. Install dependencies</p>

```
npm install
```

<p>3. Build</p>

```
npm run build
```

<p>4. Preview</p>

```
npm run preview
```

<p>5. Form API KEY</p>

```
VITE_API_KEY= 'YOUR_API_KEY' (optional)
```
